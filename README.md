# Template-rmarkdown

A change of the theme Material of Library rmdformarts in R

For change the color the theme you have to put this custom.css in the same paste of the .rmd document ,and put this in Header :

########################
# title: "Your Title"
# author: "Thalis"
# date: "28/03/2000"
# output: 
#    rmdformats::material:
#     highlight: kate
#     thumbnails: False
#     gallery: True
#     code_folding: show
#     css: custom.css
#    
# pkgdown:
#   as_is: true
